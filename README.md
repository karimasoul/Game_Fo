# Tower Defense – Système de Pathfinding

Petit jeu Tower Defense développé sous Unity qui combine deux algorithmes d’IA :

- A* pour le déplacement stratégique des ennemis

- Dijkstra pour établir les chemins des alliés
  

Technologies utilisées:

- Unity (C#)

- A* Pathfinding

- Dijkstra Pathfinding

- Architecture basée sur une grille (Grid-based)
